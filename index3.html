Rpg Maker

In Mr.Smith's class we are currently Creating a game using RPG Maker. Mr.Smith will give us a prompt such as making an object that is required to push away large boulders and we will have to make the item and make the code in the game so anytime that boulder is found in the game we will need that required item to push it away

Maker can be used to make many different types of rpg games with different mechanics and everything. It all just depends on the preference of the creator of that game.

In our game the end goal is for us to eventually make it to the school after going through many different challenges that we have coded into the game such as a boulder pass where you would have fought your first boss, a lost forest where you will have to find your way out and defeat a chicken "boss"
